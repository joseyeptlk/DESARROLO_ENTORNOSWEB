Todas los turnos y horarios se encuentran disponibles de manera 100% online. Algunos de nuestros cursos tiene convenios con Bootcamps de alto nivel y con la Alianza Ingresa de Reino Unido. Recuerda que para acceder a nuestras clases virtuales, necesitas una laptop con conexión a Internet.

Curso

TOEFL IBT| Internet
Based Test of English
as a Foreign Language

Este curso virtual está dirigido a personas que se preparan para rendir el examen TOEFL iBT, uno de los requisitos indispensables para ingresar a universidades de Estados Unidos y también para cualquier persona que desea mejorar el dominio del idioma inglés.

Plataforma: Zoom

Requisitos: Se recomienda tener conocimientos de inglés del nivel B2 (según el CEFR)

Inicio: Todos los meses (A pedido)

Duración: 36 hr

Precio: S/ 980

Curso

GRE Verbal

Este curso prepara al estudiante en las habilidades evaluadas en las secciones Verbal y Redacción del GRE. Se aprenderá a reconocer las técnicas apropiadas para resolver las diferentes preguntas en el examen oficial. Se desarrollarán prácticas intensivas aplicando estrategias adecuadas para incrementar el nivel de vocabulario y mejorar el área del pensamiento crítico.

Plataforma: Zoom

Requisitos: Se recomienda tener conocimientos de inglés del nivel B2 (según el CEFR)

Inicio: Todos los meses (A pedido)

Duración: 36 hr

Precio: S/ 890
